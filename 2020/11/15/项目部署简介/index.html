三、部署一个爬虫项目(最基础的方式)

3.1 普通部署一个 scrapy 项目

​ 1、安装好服务器系统
2、安装python3.6
3、把爬虫项目拷贝到目标服务器
4、运行，会提示缺少module
5、安装对应的模块，通过在开发环境下执行
pip freeze > requirements.txt
会在cmd命令窗口的当前目录下生成一个 requirements.txt 文件
内容类似：
aiodns==1.1.1
aiohttp==3.1.0
amqp==2.2.
asn1crypto==0.24.02

    是所有当前环境下的第三方库和对应版本

    将这个 requirements.txt 拷贝到服务器下
    执行：
    pip install -r requirements.txt
    会自动安装txt 文本中的所有第三方对应版本的库
6、继续运行，成功
