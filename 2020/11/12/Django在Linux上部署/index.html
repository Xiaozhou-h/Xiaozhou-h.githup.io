Django在Linux上部署

Veröffentlicht am 2020-11-12 | Comments:

1.一个完整的Linux系统此处略

2.安装python

// 安装Linux的wget工具，用于在网上下载文件

yum -y installl wget

// GCC编译器环境，安装python 3 时所需的编译环境

yum -y install gcc

// Python 3 使用的依赖组件

yum install openssl-devel bzip2-devel expat-develgdbm-devel

yum install readline-develsqlite*-develmysql-devellibffi-devel

// 正式安装python（因为是从官网下载，所以会比较慢）

wget “https://www.python.org/ftp/python/3.7.3/Python-3.7.3.tgz"

// 解压Python-3.7.3.tgz，并进入python-3.7.3文件夹

tar -zxvf Python-3.7.3.tgz

cd Python-3.7.3

// 依次输入编译指令，并进入python

sudo ./configure 或者 ./configure –enable-optimizations

sudomake sudomake install (感觉没啥用)
